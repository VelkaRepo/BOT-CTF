# Images

- Open file and look
- Run `file`
- Run `strings`
- Run `binwalk`, if anything extract it using `binwalk -e`
- Run `exiftool`
- Open in text editor check start/end for markers
- If .gif run `convert`
- Run stegsolve.jar
- Check LSB
- You can also use online tools [Forensically](https://29a.ch/photo-forensics/#jpeg-data)
